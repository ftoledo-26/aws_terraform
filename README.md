# Ejercicios de Terraform con AWS

Repositorio de prácticas y ejercicios de Terraform para despliegue de infraestructura en AWS.

## Estructura del Repositorio

- **Clase/** - Ejercicios básicos de clase
- **EjercicioClase/** - Ejercicios prácticos
- **Bastion_Front_Back/** - Arquitectura con Bastion, Frontend y Backend
- **PilaLAMP/** - Despliegue de pila LAMP (Linux, Apache, MySQL, PHP)
- **HostVirtuales/** - Configuración de hosts virtuales
- **Route53/** - Ejercicios con DNS en Route53
- **execirse_PreExamen/** - Ejercicios de preparación para examen

## Comandos básicos de Terraform

```bash
# Inicializar el directorio de trabajo
terraform init

# Ver el plan de ejecución
terraform plan

# Aplicar los cambios
terraform apply

# Destruir la infraestructura
terraform destroy
```

## Requisitos

- Terraform instalado
- AWS CLI configurado con credenciales válidas
- Cuenta de AWS activa
